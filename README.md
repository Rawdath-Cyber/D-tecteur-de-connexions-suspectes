# D-tecteur-de-connexions-suspectes
Ce projet Python simule un outil simple de surveillance en cybersécurité.
Le programme analyse une liste d'utilisateurs connectés et détecte les comptes privilégiés tels que :
admin
root
Lorsqu'un compte privilégié est détecté, une alerte est affichée.
Concepts utilisés
Variables
Listes
Boucles for
Conditions if / else
Opérateurs logiques
Objectif pédagogique
Ce projet a été réalisé dans le cadre de mon apprentissage de Python et de la cybersécurité.
